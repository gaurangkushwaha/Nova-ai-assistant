🤖 Nova AI Voice Assistant

A Python-based voice assistant that performs tasks like opening websites, playing music, reading news, and answering queries using AI.

✨ Features
🎙️ Voice Commands
🧠 AI Responses
🔊 Text-to-Speech
🌐 Website Automation
🎵 Music Playback
📰 News Updates
🛠️ Technologies

Python, OpenAI API, SpeechRecognition, gTTS, pygame, requests
